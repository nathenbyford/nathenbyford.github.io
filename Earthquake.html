USGS Earthquake Project

What is this project?

This is a great question, and frankly I'm still trying to figure that out myself. So far I know that I want to gain some experience using data from an Application Programing Interface(API) and maybe build a dashboard or shiny application with it.

So this project is just a way for me to use the skills I have been learning and apply them to real data. I hope to be able to learn more through the process and also learn more about earthquakes too. Currently the end goal is a dashboard that shows a map with points for earthquakes from the last day or any day that is inputed.

The Data

Like I stated in the previous sectionthis data comes from an API so I could gain some experience with that. The data comes from the USGS earthquake catelogue API in particular. Having the data come from this API there are plenty of resources on their page about what data is included. Even still after getting the data into R There are still some variables I am not sure what they are.

I decided for the initial stages of the project such as EDA and training (potentially), I would only use one week of data from the API and not many years worth of data. This made it easy for me to test out different things on a smaller set of data. The week I got the data from was Jan. 9 to Jan. 16, 2022.
