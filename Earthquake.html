USGS Earthquake Project

What is this project?

This is a great question, and frankly I'm still trying to figure that out myself. So far I know that I want to gain some experience using data from an Application Programing Interface(API) and maybe build a dashboard or shiny application with it.

So this project is just a way for me to use the skills I have been learning and apply them to real data. I hope to be able to learn more through the process and also learn more about earthquakes too. Currently the end goal is a dashboard that shows a map with points for earthquakes from the last day or any day that is inputed.
